# Examples

## `demo-import.json`

Simulated StreakGrid backup (~12 weeks ending 2026-07-20) with seven habits, rest days, notes, and mixed completion so Today / Analytics / streak maps look populated.

**Import:** open the live app → Settings → **Import JSON** → choose this file.

Import **replaces** the browser’s current StreakGrid data. Export a backup first if you already have real habits. If Drive sync is on, the next push can overwrite the Drive file with this demo; disconnect or export first if that matters.
